# 🍽️ Koya Restaurant (Frontend)

A modern restaurant web application built with React that allows customers to browse the menu, place food orders, manage their shopping cart, and enjoy a responsive user experience across desktop and mobile devices. The frontend communicates with a Django REST Framework backend via REST APIs.

## 🚀 Live Demo

https://koya-restaurant.vercel.app/

## ✨ Features

- Browse restaurant menu
- View food categories
- Shopping cart functionality
- User authentication
- Place food orders
- Responsive design
- REST API integration with Django REST Framework

## 🛠️ Tech Stack

- React
- JavaScript
- HTML5
- CSS3
- React Router
- Fetch API

## ⚙️ Installation

```bash
git clone https://github.com/Olukoyataiwohammed/Koya-restaurant.git

cd Koya-restaurant

npm install

npm start
```
